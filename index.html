arduino-hero

A midi adapter for any guitar hero controller!

This works for pretty much any GH or RB guitar controller out there, as it just

intercepts the button presses and the potentiometer. You can then send the signal

through the arduino's usb via serial to midi program (I recommend Hairless and

loopMIDI), or through an installed midi port. Please note that I have only tested on my computer. If it isn't working, the baud rate may need to be changed to work with a midi port.

Can play in a range of 32 notes (just under 3 octaves) through increasingly intense fret progressions. In general, the more fret buttons held down, the higher the note.

The code is (hopefully) straightforward enough for you to change things if you like. Check out the howtoplay file for the chord map, or check out this thread.

Supported features:

  • Whammy bar
  • Chord mode (major/minor)

  • Tuning up/down your base octave

  • Hairless Serial to MIDI

  • LoopMIDI internal MIDI driver
